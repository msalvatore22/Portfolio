The Daily Coder

Node.js | Express | React | Redux | MongoDB | PassportJS | Axios | Materialize | News API

The daily coder is a web-application for developers to stay up-to-date on the latest technology trends and news. With a Node.js/MongoDB back-end API and React/Redux front-end, users can search news topics and bookmark their favorite articles. This project is my first full-stack application that I deployed after graduating bootcamp in the summer of 2018.
